# cPager-phonegap-boilerplate


## Install

    Install Node.js
    Install Phonegap [ $ npm install phonegap -g ]
    Clone repository
    
## Develop / Serve

    $ cd path/to/app
    $ phonegap serve

### Libraries / Frameworks

    - Phonegap v7.1.1  - Apache-Lizenz 2.0
    - jQuery v3.2.1 - MIT
    - cPager v0.4.2 - MIT
